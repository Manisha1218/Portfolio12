# 🚀 Portfolio12

A modern animated developer portfolio website built with **React**, **Vite**, **Tailwind CSS**, and **GSAP**.
Designed with smooth animations, interactive UI, responsive layouts, and a premium modern aesthetic.

---

## ✨ Features

* 🎨 Modern UI/UX Design
* ⚡ Smooth GSAP Animations
* 📱 Fully Responsive Layout
* 🖥️ Interactive Project Showcase
* 🧩 Animated Services Section
* 📬 Contact & Social Media Section
* 🌙 Dark Theme Styling
* 🚀 Fast Performance with Vite

---

## 🛠️ Tech Stack

### Frontend

* React.js
* Vite
* Tailwind CSS
* GSAP
* React Scroll
* Iconify

## ⚙️ Installation

Clone the repository:

```bash
git clone https://github.com/your-username/Portfolio12.git
```

Move into the project folder:

```bash
cd Portfolio12
```

Install dependencies:

```bash
npm install
```

Start development server:

```bash
npm run dev
```
